# di-test

> Work in progress.

Ce repository contient les modules de la librairie DI.

## Modules

### `di-core`

Coeur du conteneur de dépendances:

- scopes hiérarchiques et multi-parents;
- résolution par `Key<T>`;
- injection par constructeur;
- providers lazy;
- `bind(...)`, `seed(...)`, `provide(...)`;
- collection de plusieurs providers.

Voir la documentation complète du module: [`di-core/README.md`](di-core/README.md).
